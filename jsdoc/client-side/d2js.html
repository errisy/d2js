(static) collect(htmlElementopt, baseData, directopt, customCollectorsopt)

发起收集函数。收集总入口。设置好数据路径(html attribute data)和收集器(html attribute collector)后，调用该函数，绘制所有相关元素。
也可使用封装了该函数的 jQuery 函数:

$(htmlElement).collect(baseData, direct, customCollectors)
Parameters:
Name Type Attributes Default Description
htmlElement HTMLElement <optional>
 document.body

html 元素，从该元素及其子元素收集。
baseData object d2js.dataset}

数据，要渲染的数据
direct bool <optional>
 false

true - 直接收集 false - 使用绝对数据路径收集
customCollectors object <optional>

自定义收集器（含管道）。如

{
    myCollector : function(el, value){return $(el).height()}
}

对 htmlElement 及其子元素有指定 collector="myCollector|s"，该函数会被套用。

(static) render(htmlElementopt, baseData, directopt, customRendersopt)

发起渲染函数。渲染总入口。设置好数据路径(html attribute data)和渲染器(html attribute renderer)后，调用该函数，绘制所有相关元素。
也可使用封装了该函数的 jQuery 函数:

$(htmlElement).render(baseData, direct, customRenderers)

Name Type Attributes Default Description

数据，要渲染的数据
direct bool <optional>
 false

customRenders object <optional>

自定义渲染器（含管道）。如

{
    myRender : function(el, value){el.innerHTML = value}
}
